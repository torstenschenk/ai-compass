## 1) Content & Wording (Copy)

**Main Header**
- The Multi-Dimensional Maturity Profile

**Sub-headline**
- A high-fidelity visualization comparing your organizational performance across 7 core dimensions against the industry average.

**Dimensions (7 axes)**
Retrieved from `dimensions` table:
- Strategy
- People
- Data
- Use Cases
- Processes
- Compliance
- Tech

**Legend**
- **Your Company**: user series (this response's dimension scores)

## 3) Technical Requirements (MVP)

**Library**
- Recharts (preferred) **or** Chart.js via `react-chartjs-2`

**Data model**
Array of 7 objects:

```json
[
    { "dimension": "Strategy", "userScore": 3.2, "benchmarkScore": 2.8 },
    { "dimension": "People", "userScore": 2.6, "benchmarkScore": 2.4 },
    { "dimension": "Data", "userScore": 2.9, "benchmarkScore": 2.6 },
    { "dimension": "Use Cases", "userScore": 2.4, "benchmarkScore": 2.3 },
    { "dimension": "Processes", "userScore": 3.1, "benchmarkScore": 2.7 },
    { "dimension": "Compliance", "userScore": 2.2, "benchmarkScore": 2.5 },
    { "dimension": "Tech", "userScore": 3.0, "benchmarkScore": 2.8 }
]
```

**Score range:** **1–5** (consistent with cluster scale)

**Data computation (backend)**

**User Scores (per dimension):**
- Query `response_items` for this `response_id`
- Join with `answers` (to get answer_level/answer_weight)
- Join with `questions` (to get question weight and dimension_id)
- Join with `dimensions` (to get dimension weight)
- Aggregate per dimension:
  - Per question: selected answer contribution × question weight
  - Sum per dimension, apply dimension weight
  - Normalize to 1-5 scale

**Peer Benchmark (per dimension):**
- Query ALL `response_items` across all `responses` in database
- Optionally filter by industry: join `responses` → `companies` → filter by `industry`
- Same aggregation logic as user scores
- Compute average per dimension across all peer responses
